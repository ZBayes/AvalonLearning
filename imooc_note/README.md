# avalon 慕课网学习

## 1 avalon是什么
特点：
- 体积小，无任何以来，体积小，自带加载器，压缩后不到50k。
- 编程体验好：爽快的编程体验，不再纠结于DOM操作。
- 兼容适配好：符合国情，兼容IE6和移动端。
- 自动化测试：独有的avalon.test模块用于单元测试。
- UI库丰富：拥有全职的团队打造的UI库，功能一应俱全。

[avalon新一代UI库：OniUI](hotelued.qunar.com)

适配性和兼容性都很好，移动端适配有avalon.modern.js
